# Project Documentation

This directory contains technical documentation for developers and agents
working on the Telegram bot powered by Claude Agent SDK.

- [ARCHITECTURE.md](ARCHITECTURE.md) - full application architecture:
  components, runtime lifecycles, data flows, security boundaries, plan mode,
  permission gate, uploads, logging, tests, and extension points.
- [tasks.md](tasks.md) - scheduled-task subsystem architecture: storage,
  scheduler/runner lifecycle, timeouts, restart recovery, access, and tests.
